Bu bot Render platformasida 24/7 ishlaydigan, business/manager uchun AI yordamchi.

## Sozlash:
1. Render'da Environment Variables qo‘shing:
   - TELEGRAM_TOKEN=<sizning_bot_token>
   - XAI_API_KEY=<x.ai_api_key>
   - ADMIN_TELEGRAM_ID=<admin_id>

2. Deploy qiling:
   - Service Type: Web Service
   - Start Command: `python ai_manager_bot_multi_turn.py`

3. Keyin `/start` buyrug‘i orqali test qiling.
